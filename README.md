# 🚀 ProMSC: Cross-Sample Prototype Matching and Multiscale Spatial Correlation Consistency 
---
## ❌ The Problem

| Challenge | Description |
|-----------|-------------|
| 💰 **Costly Annotations** | Pixel-level defect labeling is expensive & time-consuming |
| 🌫️ **Low Contrast** | Industrial images make defects hard to distinguish |
| 🤔 **Class Ambiguity** | Pixels near boundaries are difficult to classify |
| 🎲 **High Variability** | Defects vary dramatically in shape, size, and appearance |
---

## ✅ Our Solution: ProMSC

> **A novel semi-supervised framework that leverages limited labeled data + abundant unlabeled data for robust defect segmentation**

### 🔧 Key Components

| Component | What It Does |
|-----------|---------------|
| 🔄 **Cross-Sample Prototype Matching** | Aligns labeled & unlabeled features at feature map level |
| 📐 **Multiscale Spatial Correlation** | Enforces structural consistency across augmentations |
| ✨ **Pseudo-Label Refinement** | Boosts prediction confidence via correlation matrix |

---

## 📊 Results

> 🏆 **Outperforms existing methods on 5 benchmark defect segmentation datasets**

✅ Handles low contrast effectively  
✅ Resolves class ambiguity  
✅ Manages extreme defect variability
